# 🚀 NLW Operator — Fullstack Project

Projeto desenvolvido durante a **Next Level Week (NLW) Operator** da **Rocketseat**, na trilha **Fullstack**.

O objetivo da aplicação é explorar conceitos modernos do ecossistema **React / Next.js**, com foco em componentes reutilizáveis, interface moderna e boas práticas de desenvolvimento.

## ⚙️ Como rodar o projeto

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/CamilleGFAlmeida/nlw-operator-2026.git
```

### 2️⃣ Acesse a pasta do projeto

```bash
cd nlw-operator-2026
```

### 3️⃣ Instale as dependências

```bash
npm install
```

ou

```bash
pnpm install
```

### 4️⃣ Rode o projeto

```bash
npm run dev
```

O servidor iniciará em:

```
http://localhost:3000
```
